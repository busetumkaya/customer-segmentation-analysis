# Customer Segmentation Analysis

## Objective:
This project focuses on customer segmentation using SQL and Python, identifying key consumer groups based on demographics and purchasing behavior.

## Dataset:
- Contains customer demographics, purchase history, and interactions
- Features: Age, Location, Purchase Amount, Time Spent on Page

## Tools & Libraries:
- SQL (Data Querying, Aggregation, Filtering)
- Python (Pandas, Matplotlib, Seaborn)

## Key Findings:
- Segmented customers based on spending behavior
- Identified high-value customers for targeted marketing
- Created visualizations to present customer insights
